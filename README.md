# Slack Bot Examples

> 1. auto-status-mod-bot

## Prerequisite

### Install Deno

https://api.slack.com/automation/deno/install 

### Install Slack CLI And Login
https://api.slack.com/automation/quickstart

## Usage

### Run Locally

```bash
$ cd auto-status-mod-bot
$ slack run
```

### Deploy

```bash
$ cd auto-status-mod-bot
$ slack deploy
```
